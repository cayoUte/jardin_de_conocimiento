## rack 
mueble donde guardamos los equipos

dentro del rack puede esta un switch (capa 2) q sirve para conectar las pc

para conectar primero se conecta patch Panel q sirve como intermediario para conectar cables utp
q tiene hilos que se conecta a particiones.

luego se hace un puente entre patch panel y el switch mediante un cable utp llamado patch cord (fabricado)
se hace el puente para no trabajar directamente sobre el equipo y solo manipular el patch panel

el jack queda conectado entre la pc dentro de la face plate y se conecta al patch panel
la distancia entre el switch a la pc debe ser menor a 100m segun la normativa.
